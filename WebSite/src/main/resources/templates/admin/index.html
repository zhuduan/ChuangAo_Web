创澳商务后台管理系统

welcome back, commander!

我们为您提供了最近系统的相关信息，请您查看并处理

通用功能页

  • 个人资料：主要供您修改自己的相关资料，如密码等
  • 待处理事件：所有的审核、通知等信息都会出现在这里
  • 帮助文档：这里将对所有功能进行详述，并详细描述所有的相关状态值

用户管理

  • 用户权限：将呈现所有注册用户的信息，以及其操作EA、以及拥有账号的相关情况
  • 用户资料：将呈现所有用户的资料，以及提供相应的处理接口（如通知、授权、修改资料、客户关怀等）

账号管理

  • 公共服务：所有适用于公共服务的账号相关信息会在这里呈现
  • EA管理：会显示所有EA相关的账号信息以及权限情况
  • 工具管理：能上传或是下载所有的小工具（包括MT4小工具、其它小工具等）

系统管理

  • 访问数据：主要统计所有的运营用访问数据，如客户活跃度、PV等
  • 日志数据：主要显示error级别异常，便于及时发现问题原因
  • 用户类资源消耗：统计了以用户为单位的资源消耗信息，作为评估系统负载和收费的依据
